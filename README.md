Loyalty Level Data Analysis Using Excel
📊 Project Overview

This project focuses on analyzing loyalty membership data using Microsoft Excel to uncover meaningful business insights through data cleaning, transformation, analysis, and visualization.

The objective was to transform raw customer loyalty data into an interactive analytical dashboard that supports data-driven business decision-making.

🎯 Business Objectives

The analysis was conducted to answer the following key business questions:

Which states have the highest and lowest number of loyalty program holders?
What is the percentage distribution of loyalty holders across states?
Which gender holds the highest number of premium loyalty memberships?
Which gender holds the lowest number of premium loyalty memberships?
How are loyalty memberships distributed across different age groups?
